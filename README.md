Cat vs Dog Image Classification using CNN
Overview

This project implements a Convolutional Neural Network (CNN) to classify images as either cat or dog. The model is trained on labeled image data and learns visual features such as edges, textures, and shapes to perform binary classification.

The project demonstrates a complete deep learning pipeline including:

Data preprocessing

Model building

Training and validation

Prediction on new images

Features

Binary image classification (Cat vs Dog)

Image preprocessing and normalization

CNN architecture using TensorFlow/Keras

Model training with validation tracking

Prediction function for custom images
